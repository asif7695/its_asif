Blog Site Project

My first full-stack web application

Overview

The Blog Site is a full-stack web application built with Flask, marking my first venture into full-stack development. This project allows users to create accounts, log in, post blogs, edit or delete their posts, and search for content. It features user authentication, a SQLite database for data persistence, and a rich text editor for creating blog posts. The project helped me understand Flask, SQLAlchemy, user authentication, file uploads, and front-end integration with Bootstrap and CKEditor.

Features

  • User registration and login with password hashing for security.
  • Create, edit, and delete blog posts with a rich text editor (CKEditor).
  • Search functionality to find posts by content.
  • User profile management, including profile picture uploads.
  • Admin dashboard for managing users and posts (restricted to admin users).
  • Responsive design using Bootstrap for a seamless user experience.
  • Database management with SQLite and SQLAlchemy for storing users and posts.

Technologies Used

Flask SQLite Jinja2 SQLAlchemy Node.js SQLAlchemy Bootstrap CKEditor
