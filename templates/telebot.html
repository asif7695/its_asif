Pookie Bot Project

A simple Telegram bot built with Python

Overview

Pookie Bot is a Telegram bot I created out of curiosity to explore how bots work on the Telegram platform. Built using Python and the python-telegram-bot library, this bot responds to basic commands like /start, /help, /about, and /contact. It was my first experiment with building a bot, and I aim to enhance it in the future to include AI-driven chat capabilities. This project helped me learn about asynchronous programming, environment variable management, and integrating with Telegramâ€™s API.

Features

  • Responds to basic commands: /start, /help, /about, and /contact.
  • Provides information about the bot and its developer.
  • Uses environment variables (via python-dotenv) for secure token management.
  • Built with asynchronous handlers for efficient command processing.
  • Simple and lightweight design, ideal for learning bot development.

Technologies Used

Python Telegram Python-dotenv python-telegram-bot
