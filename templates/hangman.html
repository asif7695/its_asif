Hangman Project

A beginner-friendly Python game

Overview

Hangman is a classic word-guessing game that I built as one of my first Python projects to solidify my understanding of basic programming concepts. The goal was to create an engaging, interactive console-based game where players guess letters to complete a hidden word within a limited number of attempts. This project helped me grasp fundamental Python skills like string manipulation, loops, conditionals, and user input handling.

Features

  • Random word selection from a predefined list.
  • Interactive console interface for letter guesses.
  • Visual representation of the hangman using ASCII art.
  • Tracking of remaining attempts and guessed letters.
  • Win/lose conditions with feedback messages.
