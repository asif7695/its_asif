Hotel Room Management System

A console-based system for managing hotel rooms and bookings

Overview

The Hotel Room Management System is a C-based console application designed to manage hotel room statuses and user bookings. It allows admins to add, update, and delete rooms, while users can book available rooms and manage their reservations. Built as a learning project, it utilizes linked lists for data management and file I/O for persistence. Future enhancements could include a GUI interface or online booking capabilities.

Features

  • Admin login to manage room operations (add, update, delete).
  • User login and registration for booking rooms.
  • View available rooms and book them with a specified date.
  • Manage reservations, including cancellation and viewing history.
  • Data persistence using text files (rooms.txt, users.txt, reservations.txt).
  • Color-coded console interface for better user experience.

Technologies Used

C/C++ Linked Lists File I/O Console Application
